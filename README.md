# VCCup2019ProblemA

Разработать один экран приложения: экран со списком документов.

Поддержать:

открытие документа (минимально: фото, текст, вкладки; максимально: все типы) на весь экран с вылетом анимации;
удаление документа;
переименование документа;
подключение VK API.
